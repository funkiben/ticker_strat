Tickerstrat's Mission

In theory, Tickerstrat has the potential to be a powerful financial simulation tool. Tickerstrat will allow users to program with a versatile scripting language to queue up trades matching technical indicators in a simulated environment. This model allows for users to test technical based models and investing strategies in real time and with no risk.

This tool is meant to be used for purely educational purposes, Tickerstrat provides no investment advice.

When can we expect it?

Tickerstrat is the creation of James Tanner and Benjamin Walls. We're doing our best to get it online as soon as possible, but this is more of a pet project than a full time job.
